# castle

Ansible playbooks to manage all the nodes.

## Features

## Usage

Install dependencies to run the playbooks.

```bash
make setup
```

Deploy the playbooks.

```bash
make deploy
```

## Development

Install development dependencies.

```bash
make setup
```

Lint the playbooks.

```bash
make lint
```

Run the tests.

```bash
make test
```
